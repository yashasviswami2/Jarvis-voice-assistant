# 🤖 Jarvis - Your Personal Virtual Assistant

![Python](https://img.shields.io/badge/Python-3.8+-blue)
![License](https://img.shields.io/badge/License-MIT-green)

**Jarvis** is a Python-based voice assistant built by **Yashasvi**. It responds to voice commands and automates everyday tasks like searching the web, telling the time, reading the clipboard, getting weather updates, and more — all using simple voice input!

## ✨ Features

- 🔍 Web search using voice
- ⏰ Tells current time and date
- 🌤 Get live weather updates
- 📢 Text-to-speech & voice recognition
- 🖥️ Open websites and desktop applications
- 📂 Perform basic file operations
- 📋 Read from the clipboard
